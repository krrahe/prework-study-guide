  • Condition before ? starts a 'either or/ true false condition', two lines seperated by a :, if true first line runs if false the second

  • An IF statement checks a condtion and will execture a task if that condition evaluates to TRUE
  • The if keyword followed by a set of parentheses () which is followed by a code block, or block statement, indicated by a set of curly braces {}.
  • Inside the parentheses (), a condition is provided that evaluates to true or false
  • If the condition evaluates to true, the code inside the curly braces {} runs, or executes
  • If the condition evaluates to false, the block wont execute.

  • IF...ELSE statements make binary decissions and execture different code blocks based on a provided conditition, We can add more then one condition
  • Comparision operators compare two values

  • the bang operator ! switches the truthiness and falseiness of a value
